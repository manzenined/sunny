# Sunny

**Sun coaching app** — personalized tanning session guidance based on UV index, skin type, and exposure history.

## The problem
Most people tan by guessing. No feedback, no awareness of cumulative exposure, no safety baseline. Sunny replaces guesswork with a coached session experience.

## What it does
- Recommends optimal tanning windows based on real-time UV data
- Coaches the user through a session with contextual guidance
- Tracks exposure history and adjusts recommendations over time
- Adapts to skin phototype (Fitzpatrick I–VI)

## Stack
React Native · Expo · TypeScript · Open-Meteo API

## Status
In active development — targeting iOS App Store launch.
